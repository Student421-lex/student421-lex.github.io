I decided to look up positioning on the four sites.

Sites I Used

  • Mozilla Developer Network
  • CSS Tricks
  • W3Schools
  • CSS Reference

Differences

One difference that I noticed right off the bat between the sites was that MDN had a more advanced (or at least it felt to me advanced) definition of the position element compared to the other sites. Another thing I noticed was different that in CSS reference it jumped straight into coding example uses for different positioning options.

A big difference I also noticed was that W3Schools had a try it yourself function after introducing a new positioning aspect. The other sites did not have something like this but they did all show examples along with text explanation.

What I Think a Code Reference Page Should Have

  • Possible definition of the code
  • Examples of Use
  • Possible link to further more indepth explanation or examples for those that are more interested than just initial use

Between the sites I didn't think there was one that was more bad than another. I thought they all explained position and gave good examples of different used of it. I would say that I do think W3Schools is the best out of the four I chose for its try it yourself function.
