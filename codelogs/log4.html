What do These CSS Properties do?

Color

Represents a color that you have chosen. Normally it would be more specific than just color and would have a description for what the color is for.

Backround-color

This is the value for what the background color of your page would be. Such as you can set it to be other than white which is the default color.

Border-color

This would change the color of a border you might have on your page. This way you can match the theme you may be going for on your page or make something stand out by giving it a border color that calls attention to it.

How Does CSS Accept Color as a Value?

  • Named

    • < named-color: blue >

  • Hexadecimal

    • Specified by #RRGGBB. Example: color:#0000ff. This would be blue.

  • RGB

    • Specified by rgb()function. Example: rgb(250,0,0). This would be red.

  • HSL

    • Stands for hue, staturation, and lightness. Example: hsl(120,100%,50%). This would be green.
