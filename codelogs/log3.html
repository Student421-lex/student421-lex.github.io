CSS Media Query

How Does it Work?

Media queries work by setting constraints, such as a minimun or max of a width, to which one wants a specific thing to happen. Such as setting the background a website to be purple when the width is wider than 350px but if it is smaller than that having the background change to default white or another color of ones choosing.

Why is it Useful?

These are useful because it is a way that one can ensure that their site will look the way they want it to depending on a relative size rule that they set.

Example Supported Media Features

  • Aspect Ratio

    • This allows you to set styles based on the aspect ration of the viewport. An example would be a <@media (min-aspect-ratio: 12/9)> and then having what you want to have show up or the page look like when its this size.

  • Resolution

    • With this one you can set a style based on a device's pixel density. An example would look like this <@media (min-resolution: 200dpi)>.

  • Width

    • This allows you to set the style based on the width of the viewport. An exampe would look something like <@media (width: 250px)>.

  • Prefers Color Scheme

    • This feature is used to look at a person's preferred theme, such as dark or light, and then asign a style that would show up specifically for that theme. It would look somthing like this <@media screen and (prefers-color-scheme: dark)> and then followed by your choice of website style.
