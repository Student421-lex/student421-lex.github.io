Difference Between Non-Semantic and Semantic Tags

The differene between Non-Semantic and Semantic tags are that semantic tags defines the content that is going inside the tags. Such as the tag <img> which is the tag for an image. This tag clearly is defining what will be inide it. A non-semantic tag such as <div> doesn't define what could be inside of it.
